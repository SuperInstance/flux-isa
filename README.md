# FLUX ISA v2.0\n\nComplete 256-opcode instruction set for the FLUX fleet-native computing ecosystem.\n\n## Files\n\n- flux_isa.json — Machine-readable ISA specification\n- flux_isa.py — Python encoder, decoder, and reference VM\n\n## Quick Start\n\n```python\nfrom flux_isa import ISAEncoder, ISADecoder, FluxVM\n\n# Encode\nbytecode = ISAEncoder.encode(0x08, 0, 1, 2)  # IADD R0, R1, R2\n\n# Decode\nopcode, a, b, c = ISADecoder.decode(bytecode)\n\n# Execute\nvm = FluxVM()\nvm.load(bytecode)\nvm.run()\n```\n\n## Stats\n\n- 247 assigned opcodes across 17 categories\n- 9 reserved opcodes\n- Fixed 4-byte instruction format\n- 16 general-purpose + 16 float registers\n\nPart of the [Cocapn Fleet](https://github.com/SuperInstance).
